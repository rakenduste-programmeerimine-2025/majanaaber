## Features

### User management

- Registration
  - email/phone verification
  - Admin approval for new registrations
  - Apartment association (link account to apartment number)
- Login
  - Account recovery
  - Failed login attempts protection (account lockout)
- Roles
  - Resident
    - Tenant
    - Owner
  - Manager
- Account Management
  - Profile management (update personal info, contact details)

### Building/Apartment management

- Manager
  - Add building and apartments
  - Assign residents to existing building/apt
  - Remove residents
  - Send invoices
  - Share documentation
  - Invoice overview and marking
    - Paid
    - Unpaid
  - Look up building resident contact information
  - Parking management (assign and track parking spaces)
  - CRUD functions for building rules & regulations repository
- Documentation
  - Announcement archive (searchable history)
  - Search functionality for documents and announcements
  - Document storage for rules, contracts, insurance
  - Meeting minutes storage

### Communication

- Manager
  - Announcement board
    - Push notifications
  - Assign bills to apartments
- Chat for whole building
  - Image sharing
  - Mute of unruly users
- P2P messaging

### Problem Reporting & Maintenance

- Issue Management
- Issue/problem reporting system
- Assignment system (assign to maintenance personnel)

- Maintenance Scheduling
  - Maintenance calendar
  - Recurring maintenance setup
  - Maintenance history tracking
- Service Providers
  - Service provider directory (contractors/companies contact info)

### Voting

- Decision Making
- Vote on building decisions
- Proposal submission (residents submit topics)
- Vote history and results
